# Brazilian E-commerce Sales Analysis (Olist) 

Hi! This is a project where I analyzed over 100k+ rows of Brazilian e-commerce data. I combined **Python** for data cleaning and **Excel** for the final interactive dashboard.

## Dashboard Preview
![Dashboard Preview](Screenshot%20(36).png)

## What did I do?
1. **The Cleaning Part (Python)**
   - Used **Pandas** to merge multiple tables and fix missing values.
   - Formatted dates and removed outliers to ensure data accuracy.

2. **The Visual Part (Excel)**
   - Built an interactive dashboard using **Pivot Tables**, **Slicers**, and **Timelines**.

## Key Business Insights
* **Seasonal Peak:** Massive spike in November 2017 (Black Friday).
* **Location Matters:** **Sao Paulo** is the largest market by order volume.
* **Revenue Drivers:** Health & Beauty and Watches & Gifts are top-performing categories.
